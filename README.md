# Nagisa Ikeda — Portfolio

AI Design Engineer homepage — minimal, builder-oriented portfolio in Next.js.

## Dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Edit content

All copy lives in `src/lib/data.ts`:

- Hero, credibility, proof lines
- Selected work (case study links point to Framer)
- Build log
- Capabilities
- Footer links (update `naraLabs` URL if needed)

## Stack

Next.js 16 · TypeScript · Tailwind v4 · Framer Motion · Geist
